# config-sh
My Linux configuration.

## To Setup Ubuntu
```bash
SCRIPT_NAME="setup_ubuntu.sh"
wget -O /tmp/$SCRIPT_NAME "https://raw.githubusercontent.com/yyang-pplus/config-sh/master/$SCRIPT_NAME" && bash /tmp/$SCRIPT_NAME

```
